About the Project

Employee Salary Management Application is a robust web application that manages employee information and calculates annual salaries by consuming REST APIs.

Technical Overview

Technology Stack:
Java 17 Spring Boot Maven Thymeleaf Bootstrap 5 REST API JUnit 5

Key Features

  • ✅ Employee data retrieval from external API
  • ✅ Annual salary calculation
  • ✅ Search employees by ID
  • ✅ Responsive design
  • ✅ Error handling and validation

  • ✅ Open-source IDE usage
  • ✅ Public GitHub repository

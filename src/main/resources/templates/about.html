About the Project

Employee Salary Management Application is a robust web application that manages employee information and calculates annual salaries by consuming REST APIs.

Technical Overview

Technology Stack:
Java 17 Spring Boot Maven Thymeleaf Bootstrap 5 REST API JUnit 5 H2 Database Spring Security

Key Features

  • Employee data management with CRUD operations
  • Real-time salary calculations and updates
  • Advanced employee search and filtering
  • Responsive and modern UI design
  • Comprehensive error handling and validation
  • Secure authentication and authorization

API Endpoints

Employee Management
  • GET /api/employees - List all employees
  • GET /api/employees/{id} - Get employee details
  • POST /api/employees - Create new employee
  • PUT /api/employees/{id} - Update employee
  • DELETE /api/employees/{id} - Delete employee
Salary Operations
  • GET /api/employees/{id}/salary - Get employee salary details
  • POST /api/employees/{id}/salary/calculate - Calculate annual salary
